Example of JS code inclusion

This project shows an example of how you can define a series of common JS includes in a file, then include those common includes in differen HTML pages.

It is based on this page: https://stackoverflow.com/questions/950087/how-do-i-include-a-javascript-file-in-another-javascript-file
